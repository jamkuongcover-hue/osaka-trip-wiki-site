# 高木批發超市三條店 (Takagi Sanjo Store) 🛒💛

IG reel 提到嘅「京都最便宜批發市場」，對應嘅就係 **高木批發超市三條店 / 卸売ひろば タカギ 三条店**。呢間係京都幾出名嘅本地掃貨點，最適合買零食、飲品、泡麵、調味料、伴手禮同日用品。

## 📍 基本資訊
- **日文名**：卸売ひろば タカギ 三条店
- **地址**：京都府京都市中京区壬生天池町35-1
- **電話**：075-822-1112
- **營業時間**：09:30 - 19:30
- **休息日**：全年無休（正月除外）
- **交通**：JR / 地鐵東西線 **二條站** 步行約 5-7 分鐘；由二條城一帶過去都方便

## ✨ 亮點
- **本地人向批發超市**：唔只遊客，京都本地人都會嚟補貨。
- **零食 / 手信集中**：餅乾、糖果、即食麵、飲品、日本限定食品選擇好多。
- **寶島特價區**：官方提到店內招牌「寶島 corner」，會有半價商品，甚至高於 7 折以上折扣。
- **兩層掃貨型店舖 vibe**：好適合集中補貨，尤其係想一次買齊手信同宵夜零食。
- **揸車友好**：自駕去補貨比市區手拎舒服，買重啲都唔怕。

## ⚠️ 注意事項
- **官方已停止退稅**：官方店舖頁已寫明自 2021/10/1 起停止免稅 / 退稅手續；網上好多舊文仲寫有 tax free，呢點要以官方為準。
- **唔建議太早掃貨生鮮**：如果買生鮮、冷凍或者好多飲品，最好安排喺比較接近回酒店 / 回程時段。
- **黃昏前入場較穩陣**：19:30 關門，建議最遲 18:30 前開始逛會舒服好多。

## 💡 Charlotte 建議點樣放入今次行程
- **最適合插入 Day 3（京都市區日）**
- 建議時段：**17:30 - 18:30**
- 原因：
  - Day 3 本身已經喺京都市中心活動，地理上最順。
  - 掃貨安排喺 **接近晚飯 / 回酒店前** 最合理，買完可以直接車返酒店，不使成日拎住行。
  - 如果嗰日錦市場已經買咗少量食品，之後去高木可以集中補手信 / 大包裝 / 飲品雜貨。

## 🗺️ 今次 itinerary 用法
- **優先用途**：手信補貨、零食掃貨、飲品 / 即食食品、價格向採買
- **不建議取代**：錦市場嘅小食 / 邊走邊食體驗
- **最佳定位**：文化行程之中的「補貨站」而唔係主景點
