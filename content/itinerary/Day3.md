# Day 3: 2026-04-27 (Mon) - 大阪往京都．古街巡禮

- **10:00 酒店 Check-out 出發去京都** 🚗
- **11:00 御金神社** (10:00 - 16:00) 💰 [[御金神社 (Mikane Jinja)]]
  - *Information*: 京都著名嘅求財神社，金色鳥居非常搶眼。洗錢儀式同求御守都好受歡迎，可以祈求財運亨通。
- **12:00 錦市場** (建議 11:00 - 15:00) 🍡 [[錦市場 (Nishiki Market)]]
  - *Information*: 四百年歷史嘅京都廚房。推薦試吓章魚蛋、厚蛋燒同各種漬物，體現京都精緻飲食文化。
  - *Must Eat Snacks*: **櫂 KAI (章魚蛋)**、**三木雞卵 (玉子燒)**、**豆乳甜甜圈**。
  - *Must Buy*: **打田漬物**、**有次廚刀**。
  - *Suggestion 1*: **[[麵屋 豬一 (Menya Inoichi)]]**
    - *Details*: 京都極具人氣嘅拉麵店，湯頭清透鮮甜，配上削成細片嘅枕崎產鰹魚乾，非常精緻。
  - *Suggestion 2*: **[[拉麵 千之風 (Ramen Sen no Kaze)]]**
    - *Details*: 湯頭濃郁而不膩，特別推薦佢哋嘅炙燒叉燒，帶有陣陣焦香味。
- **14:00 岡崎神社** (9:00 - 17:00) 🐰 [[岡崎神社 (Okazaki Jinja)]]
  - *Information*: 鄺太指定景點。境內佈滿可愛嘅兔子石像，係著名嘅求子、求姻緣同保佑安產嘅聖地。
- **15:00 祇園 / 花見小路 / 八坂神社** ⛩️ [[Gion-Area]] | [[八坂神社 (Yasaka Jinja)]]
  - *Information*: 傳統京都風情街區，充滿歷史感嘅花街。八坂神社係京都嘅總鎮守，傍晚燈籠亮起時最有韻味。
- **16:00 高台寺 / 二、三年坂** 🏮 [[高台寺 (Kodaiji)]]
  - *Information*: 通往清水寺嘅坡道，兩旁傳統建築林立，極具歷史感。二、三年坂有好多特色茶屋。
  - *Must Buy*: **七味家本舖 (七味粉)**、**西湖 (和菓子)**、**京都吸油面紙**。
- **17:00 清水寺** (6:00 - 18:00) 🏯 [[清水寺 (Kiyomizudera)]]
  - *Information*: 世界文化遺產，京都最古老寺院。清水舞台由木頭架設，從呢度遠眺京都市區景色絕佳。
- **19:00 Dinner** 🍽️ 
  - *Suggestion 1*: **[[三嶋亭 本店 (Mishima Tei Honten)]]**
    - *Details*: 創業 140 幾年嘅壽喜燒名店，喺百年日式建築入面食頂級黑毛和牛，係京都最極致嘅和食體驗。
  - *Suggestion 2*: **[[祇園牛禪 (Gion Gyuzen)]]**
    - *Details*: 位於祇園會館，提供多種牛肉部位嘅壽喜燒、涮涮鍋同燒肉放題，性價比極高。
