# 阿倍野展望台 HARUKAS 300

![[harukas300.jpg]]

- **地址**: 大阪市阿倍野區阿倍野筋1-1-43
- **營業時間**: 09:00 - 22:00 (最終入場 21:30)
- **介紹**: 阿倍野展望台位於日本第二高樓「阿倍野海闊天空大廈」的頂層（58-60樓）。展望台高度達300公尺，擁有360度無死角的透明玻璃觀景台，可以將大阪市區全景盡收眼底。58樓設有露天空中庭園，環境十分舒適浪漫。
- **Charlotte's Tip**: 建議喺日落前半個鐘上去，咁就可以一次睇晒日落同埋超靚嘅大阪城市夜景！記得去 58 樓嘅露天廣場，坐喺度吹下風飲杯嘢真係好 chill。仲有，**千祈唔好錯過 59 樓嘅廁所**！嗰度被譽為「大阪景觀最好嘅廁所」，有一面全透明嘅落地大玻璃，可以邊欣賞 300 米高空嘅絕景邊...「解決問題」，真係人生一大奇觀。😂

### Features
- **360度觀景台 (60樓)**：全透明地板及牆面，震撼感十足。
- **高空廁所 (59樓)**：全大阪甚至全日本景觀最震撼嘅廁所，必去打卡。
- **空中花園 (58樓)**：露天設計，設有咖啡廳，適合放鬆。
