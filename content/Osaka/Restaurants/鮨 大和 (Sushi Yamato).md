# 鮨 大和 (Sushi Yamato)

## 📋 基本資訊
- **地區**: 心齋橋 / 東心齋橋
- **地址**: 大阪府大阪市中央區東心齋橋 1-16-19 日宝周防町会館 1F
- **Google Map**: [鮨 大和](https://www.google.com/maps/place/%E9%AE%A8+%E5%A4%A7%E5%92%8C/@34.6721863,135.5024099,17z/data=!3m1!4b1!4m6!3m5!1s0x6000e7002a35939b:0x9b91e45ee12310c5!8m2!3d34.6721863!4d135.5024099!16s%2Fg%2F11vzvk2g0t?entry=ttu)

- **參考貼文**: [takumin8888 IG post](https://www.instagram.com/p/DVxWu26kZbO/?igsh=MWZub2Fsb2QzdHoxaQ==)
- **營業時間**: 17:30 - 22:00（官方細分兩輪：17:30-19:30 / 20:00-22:00）
- **定休日**: 官方站寫年中無休；出發前仍建議再確認

- **訂位方式**: 完全預約制（TableCheck）
- **座位**: 8席，主要為 counter
- **開業時間**: 2024-06-03

## ✨ 初步印象
- IG 文案形容為 **「大阪で月1は行きたくなるお寿司屋さん」**，即係值得翻訪嘅心齋橋壽司店。
- 係 **按當日進貨調整菜單** 嘅輕鬆 Omakase / course 路線，有 seasonal flexibility。
- 位於 **東心齋橋**，位置對 Jam 大阪行程幾方便，食完可順手行心齋橋 / 難波。
- 價位比一般板前壽司親民得多，賣點唔係 ultra-luxury，而係 **高 CP 值、輕鬆但有誠意、適合 mark 低做大阪候選餐廳**。

## 🍣 貼文提到嘅亮點
- 特別 course 曾出現：**蛍烏賊茶碗蒸し**、**中とろ**。
- 貼文語氣顯示店主人氣好高，餐廳而家似乎已經變成 **預約幾旺** 嘅繁盛店。
- 視覺上屬於精緻板前壽司路線，賣相幾吸引，值得保留做 Osaka trip restaurant reference。

## Charlotte 貼士
- 呢間喺 **心齋橋**，如果 Jam 想食一餐靚啲但唔想即刻上超高價 omakase，佢幾有機會係 **好順手嘅晚餐位**。
- 以而家掌握資料，佢最大賣點係：**位置方便、完全預約制、價錢相對親民、IG / Google / Tabelog 口碑唔差**。
- 如果你大阪想保留一間 **壽司 backup / casual-upgrade option**，我會覺得佢係值得留低。
- 但如果你想追求真係頂級 / 極高端 / 紀念日晚餐級數，呢間未必係最終王者，會比較似 **高 CP 值候選**。
